# あいうえバトル ルール仕様書（確定版）

## ゲーム概要

お題に沿った「ひらがなの言葉」を秘密裏に設定し、50音から1文字ずつ攻撃して相手の言葉を暴くワード推理ゲーム。全文字を当てられたプレイヤーから脱落し、最後まで残った人が勝利。

---

## 1. ゲーム準備

### 1.1 基本設定
- プレイヤー人数：2〜6人
- 各プレイヤーの手札スロット：**8枠**
- お題を1つ決める（例：「ポケモン名」「飲みもの」「動物」等）

### 1.2 お題設定者の決定

| 条件 | お題設定者 |
|------|-----------|
| 初回（第1ゲーム） | ランダムで1人を選出 |
| 第2ゲーム以降 | 前ゲームの最下位プレイヤー（最も早く脱落した人） |
| 最下位が同率脱落 | その中からランダムで1人 |

### 1.3 単語の登録
- お題に沿った **2文字以上8文字以下** の単語を登録する

### 1.4 表記変換ルール
- すべて**ひらがな**で登録する
- **濁点（゛）は除去**する（が→か、ざ→さ、だ→た、ば→は 等）
- **半濁点（゜）は除去**する（ぱ→は、ぴ→ひ 等）
- **小文字は大文字に変換**する（っ→つ、ゃ→や、ゅ→ゆ、ょ→よ）
- **長音（ー）はそのまま**使用する
- 余ったスロットには **✕** を入れる

### 1.5 表記変換の例

| 元の単語 | 変換後（8枠） | 備考 |
|----------|---------------|------|
| ピカチュウ | ひかちゆう✕✕✕ | 半濁点除去＋小文字変換 |
| パイナップル | はいなつふる✕✕ | 半濁点除去＋小文字変換＋濁点除去 |
| ドッジボール | とつちほーる✕✕ | 濁点除去＋小文字変換＋長音維持 |
| ビール | ひいる✕✕✕✕✕ | 濁点除去 |
| コーラ | こーら✕✕✕✕✕ | 長音維持 |

### 1.6 手番順
- 毎ゲーム、ランダムで決定

---

## 2. ゲーム進行

### 2.1 手番の流れ

1. 手番プレイヤーは**50音表から1文字を指定**する
2. 指定した文字は**ボード上で使用済みとしてマーク**する（同じ文字は2回指定できない）
3. **全プレイヤー**（自分含む）の手札を判定し、該当する文字があれば**オープン**する
   - 同じ文字が複数スロットにある場合は**すべてオープン**する

### 2.2 攻撃の連続ルール
- 指定した文字が**他プレイヤーにヒット**した場合 → **もう1回だけ追加攻撃できる**
- **1手番で最大2回攻撃**（2回目はヒット/ミスに関わらず手番終了）
- 指定した文字が**誰にもヒットしなかった**場合 → 手番終了、次のプレイヤーへ

### 2.3 自爆ルール
- 自分が指定した文字が**自分の手札にもあった場合**、自分もオープンしなければならない（＝自爆）
- **自爆のみ（他プレイヤー無傷）の場合** → 連続攻撃なし、手番終了
- **自爆＋他プレイヤーにもヒットの場合** → 連続攻撃あり（他プレイヤーへのヒットとして扱う）
- **自爆により自分が脱落した場合** → 連続攻撃の権利が残っていても手番終了

### 2.4 言葉の宣言禁止
- 途中で相手の言葉が推測できても**口に出してはいけない**
- 必ず**1文字ずつ攻撃**する（言葉そのものを当てる宣言は不可）

---

## 3. 脱落・勝利条件

### 3.1 脱落条件
- **✕以外の全てのスロットがオープン**されたプレイヤーは**脱落**する（✕スロットは判定対象外）
- 脱落したプレイヤーは以降の攻撃に参加できない（手番をスキップされる）
- 1回の攻撃で複数人が同時に脱落した場合は**同率脱落**として扱う

### 3.2 勝利条件
- 最後まで**1枚でも文字が隠れたまま残っている**プレイヤーが**勝利**
- 自分以外の全員が脱落した時点で**勝利確定**（以降の自爆では脱落しない）
- 残り1人になった時点で**即ゲーム終了**（連続攻撃が残っていても実行しない）

### 3.3 同時脱落の処理
- 最後に残ったプレイヤーが同時に脱落した場合 → **その攻撃を行ったプレイヤーの勝利**（引き分けなし）

## 4. 50音ボード仕様

### 4.1 ボード構成（攻撃対象：48文字）

| | あ段 | い段 | う段 | え段 | お段 |

| あ行 | あ | い | う | え | お |
| か行 | か | き | く | け | こ |
| さ行 | さ | し | す | せ | そ |
| た行 | た | ち | つ | て | と |
| な行 | な | に | ぬ | ね | の |
| は行 | は | ひ | ふ | へ | ほ |
| ま行 | ま | み | む | め | も |
| や行 | や | | ゆ | | よ |
| ら行 | ら | り | る | れ | ろ |
| わ行 | わ | | を | | ん |
| 記号 | ー | | | | |

### 4.2 ボードの状態管理
- 各文字は「未使用」または「使用済み（攻撃済み）」の2状態
- 使用済みの文字は再度指定できない
